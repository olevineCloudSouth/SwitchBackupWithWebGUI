How to Use the Backup Viewer

This page is designed for comparisons and pulling any necessary data. Follow the steps below to use the Backup Viewer effectively:

  • First, select the switch you want to examine and the type of data you want to analyze. These options are located at the top center of the page.
  • Next, choose the date and time for the old data you want to view.
  • For the new data, either select a date and time or check the checkbox to pull live data from the switch. Note that live data retrieval takes longer as it directly interacts with the switch.
  • If you select "last" in the time options for either backup data, the system will use the latest backup from the chosen date.
  • Click the Go button to retrieve the relevant data for both sides.
  • Press the Compare button below to generate a changelog of the differences between the two selected data points.
